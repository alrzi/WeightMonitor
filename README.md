# 🏋️ WeightMonitor

Трекер веса
---

## 🚀 Первый запуск

### 1. Установи Mise
brew install mise

### 2. Всё остальное сделает само
mise install
mise run check

✅ Готово. Открывай `WeightMonitor.xcworkspace`

---

## 🔹 Единственные команды которые тебе понадобятся

| Команда | Описание |
|---|---|
| `mise run check` | ✅ Главная команда. Полная проверка всего перед коммитом |
| `mise run g` | Быстро перегенерировать проект |
